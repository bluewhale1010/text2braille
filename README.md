Text 2 Braille
=========

A small library to convert text to braille notation.

## Installation

  `npm install @shaaslam/text2braille

## Usage
    var toBraille = require('@shaaslam/text2braille');

    var braille = toBraille('2944B');
  
  
  Output should be `⠼⠃⠼⠊⠼⠙⠼⠙⠠⠃`


## Tests
